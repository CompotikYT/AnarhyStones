AnarhyStones is a high-performance, block-based land protection plugin for Minecraft servers. Built to work seamlessly with WorldGuard, it allows players to protect their builds simply by placing a specific block.

✨ Features
Custom Protection Blocks: Create unlimited tiers of stones via .toml config files.

Visual Holograms: Automatic armor-stand holograms showing owner name and region radius.

WorldGuard Integration: Automatically applies pre-configured flags (PVP, TNT, etc.) to new regions.

Anti-Grief Physics: Protection blocks are immune to piston movement and explosion damage.

Teleportation System: Includes /as home and /as sethome with configurable warm-up delays.

Dynamic Limits: Set region limits per player or rank using permission nodes (e.g., anarhystones.limit.5).

Multilingual: Native support for English, Ukrainian, and Russian.

📋 Commands
- /as get <alias> — Obtain a protection block.

- /as toggle — Enable/disable region creation on block placement.

- /as info — View details of the region you are standing in.

- /as home — Teleport to your region's home.

- /as sethome — Set a teleport point for the region.

- /as add/remove <player> — Manage region members.

- /as count — List all your owned regions.

- /as reload — Reload configurations (Admin).

🛠 Installation
Ensure you have WorldGuard and WorldEdit installed.

Drop AnarhyStones.jar into your plugins folder.

Restart the server.

Configure your unique blocks in /plugins/AnarhyStones/blocks/.
